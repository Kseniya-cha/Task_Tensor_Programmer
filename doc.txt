В решении используются библиотеки openpyxl для чтения файлов xlsx и os для получения всех файлов директории.

Критерий успешности сотрудников - сумма разностей между фактическими человеко-днями и планируемыми для всех проектов этого сотрудника.

Предполагается, что структура файлов та же, что в примере задания.

Класс File содержит методы для обработки одного файла. Функция success_by_one_file обрабатывает один файл. На вход принимается название файла с расширением xlsx.

Последние две функции обрабатывают все файлы из директории. Успешности сотрудников, упомянутых более чем в одном файле, складываются. В результате идёт сортировка сотрудников по их успешности из всех доступных для обработки файлов. ФИО сотрудников выводятся построчно.

Реализована проверка корректности данных в ячейках с человеко-часами (числовые данные или None) и ячейках с ФИО сотрудников (должны иметь формат "Фамилия И.О.", а также данные в двух соседних ячейках, относящихся к одному сотруднику, должны совпадать). Если по какому-либо критерию данные в ячейках с ФИО сотрудников не считаются корректными, выводится сообщение об этом и указание на координаты обеих ячеек и данные, содержащиеся в них. Если открыт один из обрабатываемых файлов, также выводится сообщение об этом.
